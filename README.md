# Website Uptime Checker

A simple Bash script to check if websites are **UP ✅** or **DOWN ❌**.  
Tested on **CentOS (VMware + PuTTY)**.

## Files
- `sites.txt` → List of websites to check
- `uptime.sh` → Script that checks each site’s HTTP status
